# チャンネルプロファイル

- 最終更新: 2026-07-06
- 作成ソース: E2Eテスト用ダミー (実在の運営者設定を模したもの)

## 1. ターゲット視聴者

- 誰が: AIを仕事に使いたいが何から始めるか分からない30-40代の会社員
- 何に困っている: 情報が多すぎて最初の一歩が決められない
- 見終わった後どうなれる: 今日中に最初の操作を1つ実行できる

## 2. チャンネルの約束 (全動画共通の立ち位置)

「未経験でも今日から手が動く」まで噛み砕く。理論の解説だけで終わる動画は出さない。

## 3. 構成の型

- フックの型: 結論の言い切り → 視聴者の痛点の名指し → この動画で分かること3点
- セクション構造: 1動画3セクション。各セクション末に次への引きを1文
- まとめとCTA: まとめは箇条書き3点まで。CTAはまとめの後に1回だけ、次動画への導線を優先

## 4. 口調・文体

- 一人称 / 視聴者の呼び方: 僕 / あなた
- 語尾・トーン: です/ます基調。断定は「〜なんです」
- 口癖・型フレーズ: 「ここからが本題です」「結論から言うと」
- 1文の長さ・テンポ: 1文60字以内。3文に1回は問いかけ
- 表記: 難読漢字はひらく。専門語は初出で言い換え
- 禁止の言い回し: 「いかがでしたか」

## 5. お手本フレーズ集 (過去台本からの原文引用)

1. 「結論から言うと、今日やることはひとつだけなんです。」
2. 「難しそうに聞こえますよね。でも、実は3分で終わります。」

## 6. NG リスト

- 言わないこと: 誇張した収益の約束
- 避ける表現: 「絶対」「100%」

## 7. 勝ちパターンメモ

- 「実演しながら話す」回は維持率が高い (仮説)

## 8. 直しの履歴 (納品後に運営者が直した点 → 次回への指示に翻訳)

- 2026-07-05: 冒頭の自己紹介を削除 → 冒頭に自己紹介を入れない。1文目から本題に入る

## 9. 納品フォーマット (台本の使い方)

- 台本の使い方: そのまま読み上げる。演出メモは【】で本文と区別
- 含めるもの / 含めないもの: セクション見出しは含める。BGM指定は含めない
